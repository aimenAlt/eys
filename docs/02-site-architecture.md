## Dynamic route principle
Content files do not automatically become pages. Astro route files generate static pages by reading content collections.

Example:
- `src/content/services/tv-mounting.md`
- generated by `src/pages/services/[slug].astro`
- output: `/services/tv-mounting/`

## URL rule
Use clean, stable, lowercase, hyphenated URLs.
Never change URLs casually after launch.
