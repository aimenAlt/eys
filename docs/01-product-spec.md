# Product Spec — EYS Website

## Business
Elevate Your Space Handyman is a local handyman / home-services operator serving Katy and West Houston.

## Positioning
Premium, organized, trustworthy, professional home services — not cheap gig handyman work.

## Website goals
1. Increase qualified calls.
2. Increase quote requests.
3. Allow direct booking only for controlled, productized services.
4. Improve local SEO through service pages, service-area pages, and project pages.
5. Turn project photos into proof assets.
6. Make EYS look more organized than local competitors.
7. Prepare for possible future brand/domain migration without damaging SEO.

## Primary conversion actions
- Call
- Text
- Request quote
- Book TV mounting / simple productized job
- Schedule assessment
- Upload photos

## First 30-day build priorities
1. Convert current single-page HTML into Astro homepage.
2. Build layout, design tokens, and component system.
3. Add service data model.
4. Add service page template.
5. Add top service pages:
   - TV mounting
   - Drywall repair
   - Ceiling fan installation
   - Door repair and installation
   - Furniture assembly
   - Painting
   - General handyman services
6. Add service-area data model.
7. Add first service-area pages:
   - Katy
   - Cinco Ranch
   - Fulshear
   - Richmond
   - Cypress
   - West Houston
8. Add project model and gallery.
9. Add booking/request-quote page.
10. Add Pages CMS config.
11. Deploy to Cloudflare preview.
12. Run SEO/performance QA.

## Operational booking rule
Direct online booking is only for fixed-scope jobs. Start with TV mounting and maybe furniture assembly.
Complex jobs go to quote request or assessment scheduling.
