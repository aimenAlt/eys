# Design System — EYS

## Visual direction
Dark hero, light body, red conversion accents.

The site should feel:
- premium
- disciplined
- masculine
- local
- trustworthy
- clean
- structured

Not:
- cheap
- cartoonish
- cluttered
- bloated
- franchise-generic

## Layout
- Max width: 1180–1280px.
- Section padding: 72–120px desktop, 48–72px mobile.
- Grid: 12-column desktop, 1-column mobile.
- Cards: clean white cards, subtle borders/shadow, red accent line.

## CTA rules
Primary CTA:
- red background
- white text
- direct action copy
- always visible above fold
- repeated throughout page

Recommended labels:
- Request a Quote
- Book TV Mounting
- Call Now
- Schedule Assessment
- Send Photos for Estimate
